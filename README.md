## Quick Start

```bash [Terminal]
npx nuxi init -t github:nuxt-ui-pro/docs
```

## Setup

Make sure to install the dependencies:

```bash
npx bun install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
npx bun run dev
```


## e2e tests

```
podman run -ti -v $PWD:/app mcr.microsoft.com/playwright:v1.39.0-jammy
npx playwright test
```
