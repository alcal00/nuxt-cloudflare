# \<ppw-fullscreen\>

A tag that let show its content fullscreen.


xxx


## showZoomButton prop

::ppw-fullscreen{showZoomButton}
    Content to display in fullscreen ...
::

## showHeader prop

::ppw-fullscreen{showHeader}
    Content to display in fullscreen ...
::

## windowOnly prop

zoom only in the window

default is `false` => full-screen zoom
